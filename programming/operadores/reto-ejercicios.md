
## 1. Calculadora de Propina y División de Cuenta

Una cuenta de restaurante es de $1250.50. Se quiere dejar un 15% de propina y dividir
el total final entre 4 personas. ¿Cuánto paga cada uno?

La solución debe funcionar para cualquier cuenta, cualquier porcentaje de propina y
cualquier número de personas (usa variables para cada uno)

## 2. Conversión de Temperatura

Tienes una temperatura en grados Celsius y necesitas convertirla a grados Fahrenheit y grados Kelvin.

Formulas:
- F = (C * 9/5) + 32
- K = C + 273.15

Donde C = grados Celsius, F = grados Fahrenheit, K = grados Kelvin
